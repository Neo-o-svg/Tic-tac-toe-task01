
# Tic Tac Toe - React

Простой проект **крестики-нолики**, реализованный на **React**.

## Описание

Это классическая игра **крестики-нолики** для двух игроков на одном устройстве. Игра отслеживает текущий ход, определяет победителя и позволяет перемещаться по истории ходов игроков.

## Особенности

* Игровое поле 3x3
* Смена игроков (X и O)
* Проверка победителя и ничьи
* Кнопки с предыдущими действиями игроков

## Технологии

* React (Create React App)
* TS
* CSS (SCCS)

## Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/Neo-o-svg/Tic-tac-toe-task-01.git
```

2. Перейдите в папку проекта:

```bash
cd tic-tac-toe
```

3. Установите зависимости:

```bash
npm install
```

4. Запустите проект:

```bash
bun run dev
```

Проект будет доступен по адресу `http://localhost:3000`.
